# abkis
Arch BTRFS KDE Install Script


## Prerequisites
- A root, efi, and swap partition
- Efi on FAT32
- Root on BTRFS

## Instructions

- `wget -L -O abkis https://github.com/jeppevinkel/abkis/tarball/master | tar xz`
- `./abkis/abkis`
Simply follow the on-screen instructions until it's complete.
- `reboot`
